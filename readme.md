# Ray Tracing

A real-time ray traicing demo using taichi language.

## **Requirement**

* Python 3
* Taichi 0.6.7 or 0.6.8

## Usage

```bash
$ pip3 install taichi
$ git clone https://github.com/moranzcw/raytracing_taichi.git
$ cd raytracing_taichi
$ python3 raytracing.py
```
